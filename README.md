# STM32 串口单字节发送项目

本项目基于 STM32 HAL 库开发，实现了通过 USART 异步串口进行单字节/自定义字节流的稳定发送功能。工程结构规范，去除了编译临时文件，适合作为嵌入式底层的通信参考模块。

## 🛠 开发环境
- **MCU**: STM32F103C8T6 *(如果是其他型号，请在此修改)*
- **IDE**: Keil MDK-ARM v5 / STM32CubeMX
- **固件库**: STM32Cube FW_F1 HAL库
- **调试工具**: ST-Link V2 / 电脑端串口调试助手

## 📂 目录结构说明
- `MDK-ARM/`：Keil v5 工程文件夹（已配置 .gitignore，仅保留核心工程配置）
- `Inc/` & `Src/`：应用层核心代码（包含 `main.c`、`usart.c` 等）
- `Drivers/`：ST 官方 HAL 库及 CMSIS 底层驱动
- `UART.ioc`：STM32CubeMX 配置文件，支持二次图形化配置

## 🚀 核心功能与技术特点
1. **波特率配置**：115200 bps, 8位数据位, 1位停止位, 无校验。
2. **底层驱动**：对 `HAL_UART_Transmit()` 进行二次封装，确保了底层字符/字节发送的稳定性。
3. **工程规范**：严格执行了嵌入式 Git 提交规范，过滤了 `.obj`、`.axf` 等无用中间文件，保持仓库轻量化。
